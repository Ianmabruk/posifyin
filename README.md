# Universal Repository

This repository contains the POSify application.

## Structure

```
universal/
├── .git/              # Git repository
├── .gitignore         # Git ignore rules
├── .nvmrc            # Node version specification
├── README.md         # This file
└── my-react-app/     # Main application code
```

## Getting Started

All application code is located in the `my-react-app/` directory.

```bash
cd my-react-app
npm install
npm run dev
```

For more details, see [my-react-app/README.md](my-react-app/README.md)
